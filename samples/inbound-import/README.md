# 采购入库识别 — 测试样例

与 `api/prisma/seed.ts` 种子数据中的物资编码一致，可直接用于测试「新建采购入库单」的识别功能。

| 文件 | 用途 |
|------|------|
| `采购入库清单-测试.pdf` | 点击 **识别 PDF** 上传 |
| `采购入库清单-测试.png` | 点击 **识别图片** 上传 |

## 预期识别结果（5 行）

| 编码 | 名称 | 数量 |
|------|------|------|
| YJ-A-006 | 防汛沙袋 | 500 |
| YJ-A-007 | 抽水泵 | 20 |
| YJ-B-002 | 12㎡应急帐篷 | 100 |
| YJ-B-005 | 救生衣 | 300 |
| YJ-C-011 | 医用口罩 | 10000 |

## 使用步骤

1. 确保已执行过 `pnpm db:seed`（物资档案存在）
2. 打开 **采购入库 → 新建采购入库单**
3. 分别上传 PDF 或 PNG 测试
4. 识别成功后明细表应自动回填，并展示编码、规格等属性
